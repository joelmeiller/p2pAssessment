System Requirements

Folgende Anforderungen an den Web-Server sollten erfüllt sein, damit die Applikation reibungslos funktioniert.

  • Es wird empfohlen mind. PHP Version 5.3.0 installiert zu haben.
  • Die PHP Installation sollte die Extension MySQLi Version 5.0.0 untersützen.
  • Datenbanksystem ab mind. MySQL Version 5.0

Setup

Es existiert kein automatisierter Installer und die Applikation muss manuell aufgesetzt werden. Es werden admin-Rechte für die MySQL-Datenbank und die Filemanipulation auf dem Web-Server benötigt.

Applikation auf dem Web-Server installieren

Zuerst müssen die dekomprimierten Originalfiles (download als zip hier), in der vorgegeben Struktur in das entsprechende Verzeichnis des Web-Server kopiert/extrahiert werden.

Danach muss die Datenbank p2p_assessment_db durch ausführen des Files db/p2p_assessment_V1.0_MySQL_DB_setup.sql erstellt und aufgesetzt werden.

DB Connection

Im File php/DBConnection.php müssen die MySQL Datenbank der geltende Benutzernamen und das Passwort gesetzt werden

 

Projekt

Der Projektname muss manuell gesetzt werden, in dem das Attribut title in der Tabelle project geändert wird.

Vorlage für UPDATE-Statement: UPDATE p2p_assessment_db.project SET title='<Titel>' WHERE id='1';

Test

Rufe die Webapplikation auf. Es sollte ein Login-Fenster erscheinen und man kann sich mit dem Admin (QM) Benutzer einloggen
