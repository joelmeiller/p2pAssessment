Benutzeranleitung

Die Applikation ist selbsterklärend. Eine detaillierte Benutzeranleitung ist daher nicht notwendig. Hier sind einige Hinweise zur Benutzung:

  • Die Daten der Bewertung werden gespeichert, sobald auf Next oder Back gelickt wird
  • Wir die Bewertung abgeschlossen, können keine weiteren Bewertungen vorgenommen werden
  • Ist die Option, dass auch Teammitglieder alle Bewertungen sehen können aktiviert, kann aus der Übersicht zusätzlich die Gesamtbewertung über alle Teammitglieder eingesehen werden.

Admin (Quality Manager)

Zusätzliche Hinweise für die Benutzung als Admin (Quality Manager)

  • Ist der Benutzer als Admin angemeldet, kann er keine Bewertung abgeben. Ist es erforderlich, dass der Quality Manager in der Rolle als Teammitglied eine Bewertung abgeben soll, dann muss ein zusätzliches Teammitglied unter Quality Manager angelegt werden.
  • Als Admin kann der Benutzer Kriterien und Teammiglieder hinzufügen, ändern und löschen.
  • Beim Hinzufügen von neuen Teammitgliedern muss initial ein Passwort gesetzt werden. Die Speichern Option ist erst verfügbar, wenn das Passwort korrekt wiederholt wurde.
